FrameNote v2.22 추가 폰트 안내
================================

기존에 앱에서 바로 사용할 수 있던 기본 폰트는 유지되어 있습니다.

추가 폰트는 두 가지 방법으로 사용할 수 있습니다.

방법 1 — 앱 화면에서 불러오기
-----------------------------
스타일 탭의 '추가 폰트 파일 불러오기'에서 보유한 폰트 파일을
한 번에 선택합니다. 브라우저 저장소가 허용되면 다음 실행에도 유지됩니다.

v2.23 폰트 로딩 최적화
======================

- 등록 시: 폰트 파일을 브라우저 저장소에만 보관합니다.
- 선택 시: 제목 / 본문 / 메타에서 선택한 폰트만 FontFace로 불러옵니다.
- 재사용: 이미 불러온 폰트는 같은 실행 중 다시 해석하지 않습니다.
- 시작 시: 저장된 모든 폰트를 복원하지 않고 현재 선택된 폰트만 불러옵니다.
- 저장 작업: 폰트 파일은 최대 3개씩 병렬 저장합니다.
- 미리보기: 필요한 폰트 로딩이 끝난 뒤 한 번만 다시 그립니다.

폰트 파일 자체는 이 배포 ZIP에 포함되어 있지 않습니다.
앱의 '추가 폰트 파일 불러오기'에서 선택하거나,
안내된 정확한 파일명으로 이 폴더에 직접 넣어 사용하세요.


v2.24
-----
사용자가 제공한 OTF/TTF 폰트 파일을 이 폴더에 포함했습니다.
앱 시작 시 전체 폰트를 즉시 로드하지 않고, 선택된 폰트만 필요할 때 로드합니다.
