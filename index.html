EXIF 직접 수정
사진에서 읽은 값을 수정하거나, 표시용 정보를 직접 입력합니다.
입력한 값은 모든 템플릿의 EXIF 표시와 브랜드 자동 판별에 바로 반영됩니다.
JPEG에 EXIF가 없거나 PNG·WebP 파일인 경우에도 표시용 정보를 직접 입력할 수 있습니다.

추가한 제조사 로고 없음
PNG · SVG · WebP · JPEG 사용 가능(최대 2MB). 투명 배경의 PNG 또는 외부 이미지를 참조하지 않는 SVG를 권장하며, 로고 색상은 파일 원본 그대로 출력됩니다. 이 기능은 이미지 로고를 표시하는 템플릿에서만 사용할 수 있습니다.

Brush Photo Edge 전용
Multi Frame 전용
Multi Frame는 원본 사진 비율을 유지하는 심플 프레임입니다. EXIF, 제목, 위치, 로고, 워터마크는 표시하지 않고 저작자만 선택적으로 표시합니다. 필요한 장수만큼 사진을 직접 넣어 사용하는 것을 기준으로 합니다.
요소별 위치 · 크기 · 행간
현재 템플릿에서 사용하는 요소만 표시됩니다. X·Y 이동 범위를 프레임 전체보다 넓게 확장해 각 요소를 프레임 안 어느 위치로든 이동할 수 있습니다. Thought·Note·질문 문구는 여러 줄일 때 행간도 조절할 수 있습니다. 저작자와 워터마크도 크기와 위치를 독립적으로 조절할 수 있습니다. 조정값은 템플릿별로 따로 저장됩니다.
Poster Overlay 전용
기준점으로 큰 위치를 잡고, 요소별 위치 카드에서 미세 조정합니다.
